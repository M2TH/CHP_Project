Documentation :


Comment compiler notre code :
ouvrir un terminal dans le dossier du projet (celui ou se trouve le fichier makefile)
taper "make" dans le terminal
Cette commande lance les commandes de compilation enregistré dans le fichier makefile.

Exécution :
entrer dans le terminal (après compilation) :
"./run"

Modification des paramètres :
Les paramètres peuvent être modifier dans le fichier "paramètre.txt"
Les paramètres y sont stocké sous la forme :
"#nom_du_paramètre
valeur_du_paramètre"
